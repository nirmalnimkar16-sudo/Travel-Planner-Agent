# AI Travel Planner Agent

## Overview
This project is an AI-powered Travel Planner Agent built using IBM watsonx Orchestrate.

## Features
- Personalized travel planning
- Destination recommendations
- Budget-friendly itineraries
- Hotel and transport suggestions

## Technologies Used
- IBM watsonx Orchestrate
- IBM Cloud
- GPT-OSS 120B
